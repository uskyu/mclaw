### 重要发现

1. **Gemini 原生 API 格式效果更好**
   - 端点: `/v1beta/models/gemini-3-pro-image-preview:generateContent`
   - 支持 `imageConfig` 参数控制分辨率和比例
   - 可生成 2K (2048×2048) 高清图片

2. **关键配置参数**
```json
{
  "generationConfig": {
    "responseModalities": ["TEXT", "IMAGE"],
    "imageConfig": {
      "aspectRatio": "1:1",
      "imageSize": "2K"
    }
  }
}
```

3. **提示词注意事项**
   - 避免使用双引号 `"` 会导致 JSON 解析错误
   - 高难度复杂排版提示词会导致失败
   - 中等复杂度提示词（200-400字符）效果最佳

4. **并发限制**
   - 3.1-flash: 支持高并发（10并发稳定）
   - 3-pro: 低并发更稳定（5并发成功，10并发超时）

### 结论
- OreAPI 适合中等复杂度的图片生成任务
- 需要高清图片时使用 Gemini 原生 API 格式
- 批量生成使用 3.1-flash，高质量单张使用 3-pro
